Will contain UML models that result from running the Eclipse workflow UMLinstantiator.
(i) Copy this folder to the folder containing the Coq/Haskell transformation, and run CompileUml2RdbmsAndCreateXsdFiles.sh and TransformAllInstantorResults.sh.
(ii) Now copy the folder from there to the folder containing the ModelMorf UML2RDBMS transformation, and run TransformInstantiatorResults.bat
Copy the resulting folder finally to the edu.kit.ipd.sdq.mdsd.rdbmscomparator project.
